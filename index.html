Kingdom Hearts 2 Custom Goals made by CJ2123

Standard

  • Standard Bingo where you select a size and generate a board with all squares revealed.
  • You can play various bingo formats such as 3 line or blackout.

    • Exploration

  • Select a size and select starting squares on the mini bingo board. You can also select goal squares.
  • If you check a square, it will reveal the goals above, below, to the left, and to the right of it.
  • Like Standard, you can also play various bingo formats in this mode.

    • Roguelike

  • When you generate a card, click the START square to begin.
  • If you check a square, you'll be able to see 3 squares on the next row. 1 to the left of it, 1 to the right, and 1 directly below it.
  • If you go to the left and right ends, only 2 squares will be revealed.
  • B8, B12, B16, B20 are forced goals. The next row will be resumed from the previous layer.
  • Check B20 to complete the board.

    • JSON

  • The board in JSON format for Bingosync.
  • If the size is 5x5 or less, it can be used in Bingosync by selecting "Custom (Advanced)" in the Game option.
